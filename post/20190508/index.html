Android开发错误集合

2019-05-08 15:00

本篇收录日常Android开发中遇到的问题以及解决思路。

  1. Android studio Program type already present: com.****.BuildConfig【2019-05-08】

【错误原因】 Android studio 抛错，是因为有2个module在 AndroidManifest.xml 里面具有一样的package name。
【解决方案】 将module中的AndroidManifest.xml中的packagename修改成不同名字即可。

2.在lib-module中使用ButterKnife需要使用R2。不能使用switch 来判断resId，必须使用if-else来进行判断。
2. 待续。。。
